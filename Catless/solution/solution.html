При подключении к контейнеру участник соревнований попадает в систему, в которой удалены основные утилиты для редактирования и вывода текстовых (и не только) файлов, т.е. такие программы, как cat, less, more, tail, head, etc. Участнику необходимо найти способ прочитать файл flag.txt. Для этого можно воспользоваться не такими распространенными утилитами, как nl (выводит файл с пронумерованными строками), cut (позволяет вырезать куски из файлов, например, сработает 'cut -b 1- flag.txt'), tac (выводит файл задом наперед), или же можно прочитать файл прямо из bash: while read line do printf "$line" done < flag.txt Вместо printf можно использовать и echo, но для этого сначала нужно сбросить псевдоним для echo ($ unalias echo), который коварно указывает на несуществующий /bin/echo.
